# Kadry Forms

Статичні кадрові анкети (форма → готові PDF-документи прямо в браузері, без бекенду).
Одне репо на всі кадрові питання наших фірм.

## Форми

- **[/](https://vladsh77.github.io/kadry-forms/)** — FAYNA DIGITAL sp. z o.o. (umowa zlecenia)
- **[/campscout/](https://vladsh77.github.io/kadry-forms/campscout/)** — CampScout: анкета кадри табору (instruktor PL + волонтер UA)

Кожна форма самодостатня (свій `pdfmake` + `vfs_fonts` + генератор), дані нікуди не надсилаються — все відбувається локально в браузері кандидата.
