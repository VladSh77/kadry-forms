Анкета кадри дитячого табору

Заповніть анкету — на її основі буде сформовано ДВА договори: волонтерський (опіка над дітьми, ТОВ КЕМПСКАУТ, Україна) та інструкторський (виконання програми табору, CampScout, 31,40 zł/год), а також окрема згода на перевірку кандидата. Займе 5–10 хвилин. Поля з * обов'язкові.

1. Дані латинкою (для польського договору)

Пишіть ТОЧНО як у закордонному паспорті (латинськими літерами) — ці дані підуть саме у польський договір з CampScout.

2. Дані українською (для українського договору)

Повне ПІБ кирилицею піде в український договір волонтера (ТОВ КЕМПСКАУТ). По батькові вказуйте лише за наявності.

3. Документи, що посвідчують особу

Для польського договору
Можна вказати або закордонний паспорт, або karta pobytu (посвідка на проживання в Польщі) — оберіть той, який маєте.
Для українського договору
Можна вказати або закордонний паспорт, або внутрішній паспорт України — оберіть той, який маєте.

4. Ідентифікаційні номери

PESEL — польський ідентифікаційний номер, потрібен для польського договору, ZUS та податків. Якщо не маєте — позначте чекбокс нижче.
Український реєстраційний номер облікової картки платника податків (ІПН). Потрібен для українського договору.

5. Адреса проживання

Вкажіть ОДНУ адресу кирилицею — вона автоматично транслітерується для польського договору, вводити двічі не потрібно. Приклад: «вул. Хрещатик 1, кв. 5, 01001 Київ».
Для польського договору автоматично:
⚠️ Якщо ви вже проживаєте в Польщі — автоматична транслітерація може спотворити назви (напр. «Познань» ≠ «Poznań»). Впишіть нижче адресу ТОЧНО польською мовою, і вона матиме пріоритет.

6. Рахунок для виплати

Вкажіть хоча б один рахунок. Якщо немає польського IBAN — достатньо вказати звичайний український рахунок.

★ 12. Згода на перевірку кандидата (безпека дітей)

CampScout, як організатор дитячого табору, ПЕРЕД допуском до роботи з дітьми перевіряє кандидатів за реєстрами осіб, які вчинили злочини проти статевої свободи та недоторканості неповнолітніх. Це стандартна практика захисту дітей.
